Each handler follows this pattern:
```typescript
server.tool("tool_name", "description", { /* zod schema */ }, async (params) => {
  try {
    const result = await domainFunction(...);
    return { content: [{ type: "text", text: JSON.stringify(result, null, 2) }] };
  } catch (e) {
    const msg = e instanceof Error ? e.message : String(e);
    return { isError: true, content: [{ type: "text", text: msg }] };
  }
});
```

2. Call `resolveAccount(accounts, account)` at top of try block
3. Pass resolved AccountConfig to domain function as first arg
4. Include `account: acct.label` in the response JSON
